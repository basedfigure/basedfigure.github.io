EmulaTotM

1.0 - Introduction:

- EmulaTotM (etotm) is a gamemaster (GM) emulator for designing and playing tabletop roleplaying games, both solo and multiplayer. It supports custom modules and is designed with BioGriever: A Dark Future in mind.

The game got inspired by the following game formats and genres, in order of historical appearance with their wikipedia pages:

  • Miniature wargames (TMGs)
  • Tabletop roleplaying games (TTRPGs)
  • Multi-user dungeons (MUDs)
  • Roguelikes (RLs)
  • Computer/Western roleplaying games (CRPGs/WRPGs, used interchangeably)

1.1 - Applications:

- etotm is cross-platform application for GNU Linux, Android, Windows and macOS -- with Linux first, as it is open source, well supported, free, easy to use, and painless to install on both old and modern computer hardware, with plenty of tutorials available so you can set it up at home. Linux originates from Finland, as an independent project, but has since been adopted for various use cases worldwide.

- The development environment is a Fedora Linux 42 Workstation - as the OS, that comes with the GNOME desktop environment. The Lazarus-IDE can be installed through the Software Center or the console (with more on each below).

2.0 - Tools & dependencies:

- Relies only on natively supplied lazarus packages, my own code and badsector's JTF model format. Mine loosely follows some 'not invented here' philosophies

  • Free Pascal (FPC) at home page
  • Lazarus-IDE at home page
  • BioGriever mechanics at codeberg
  • Waistworks 3d units at codeberg
  • WebWizEd html units at codeberg

2.1 - Fedora Linux packages:

  • - Lazarus-IDE can be downloaded and installed through the Software Center, or through a console with:
  • - sudo dnf install lazarus-ide
  • - LazOpenGLContext installation requires mesa-libGL-devel to be present, which can be installed with:
  • - sudo dnf install mesa-libGL-devel

2.2 - Lazarus packages:

  • - Installed through Lazarus, from Package->Install/Uninstall Packages ...

2.3 - Ptyxis console on Lazarus:

  • - To use the Fedora's default console Ptyxis on Lazarus, you need to set the following parameters in Run/Run Parameters:
  • - [x] Use launching application
  • - ptyxis -e bash -c "$(TargetCmdLine); echo; echo Press enter to exit; read"
